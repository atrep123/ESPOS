# Security & Analytics Reports

This directory contains auto-generated reports from CI/CD workflows.

## Security Dashboard

**File**: `security_dashboard.md`

Auto-generated by `tools/security_dashboard.py` during the weekly security audit workflow.

**Contents**:

- SARIF findings summary (pip-audit, npm audit, Bandit, CodeQL)
- Vulnerability counts by severity
- License violations
- SBOM component counts
- Timestamp of last update

**Update Frequency**: Weekly (scheduled) + on-demand (manual workflow trigger)

**How to Read**:

- **Errors**: HIGH/CRITICAL findings requiring immediate action
- **Warnings**: MODERATE findings to review
- **Notes**: LOW/INFO findings for awareness

## Performance Reports

**Files**: `perf_trend.csv`, profiler artifacts

Generated by performance profiling tools during CI runs.

## Test Artifacts

**Files**: `ci_smoke_*.log`, `*_test_scene.html`, `*_preview_*.png`

Snapshots and logs from automated UI tests and simulator runs.

## Release Artifacts

**File**: `RELEASE_ARTIFACTS.md`

Summary of files included in GitHub Releases (checksums, build variants).

*Note*: Files in this directory are gitignored except for `README.md`, `RELEASE_ARTIFACTS.md`, and `security_dashboard.md`.
