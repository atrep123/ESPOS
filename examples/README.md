# Examples

This folder contains demo artifacts generated by the UI tools and tests:

- ui_demo.json/html/png (from `ui_export_c.py`)
- dashboard_demo.json/html/py (from `test_ui_interactive.py`)
- showcase.json/html/py (from `test_showcase.py`)

Regenerate them via:

```powershell
python ui_export_c.py
python test_ui_interactive.py
python test_showcase.py
```
