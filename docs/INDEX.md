# Documentation Index

Central hub for key development and usage docs of the ESP32 UI Simulator + Designer.

## Quick Start Paths
- **Rapid Onboarding**: `QUICKSTART.md` – install, run, essential tasks.
- **Simulator Usage**: `SIMULATOR_README.md` – runtime parameters, RPC/UART, layout, troubleshooting.
- **UI Designer Guide**: `UI_DESIGNER_GUIDE.md` – workflow, commands, templates, export.
- **Examples & Integrations**: `SIMULATOR_EXAMPLES.md` – extended scenarios.

## Testing & Verification
- **Smoke Script**: `tools/ci_smoke.ps1` – fast local verification.
  ```powershell
  powershell -NoProfile -ExecutionPolicy Bypass -File .\tools\ci_smoke.ps1
  ```
- **Full Tests**: `Tests: Run All` VS Code task or:
  ```powershell
  python -m pytest -q
  ```
- **Designer Specific**: See `UI_DESIGNER_TEST_RESULTS.md` for historical outcomes.

## Performance & Profiling
- **High-Level Report**: `PROFILER_REPORT.md` – summary of metrics and methodology.
- **Advanced Expansion**: `PERFORMANCE_PROFILER_EXPANSION.md` – deeper dives & future ideas.
- **Collect Metrics** (timing CSV):
  ```powershell
  python sim_run.py --export-metrics metrics.csv --fps 144 --width 64 --height 16
  ```
- **Analyze Metrics**:
  ```powershell
  python performance_profiler.py metrics.csv profiler_enhanced.html
  ```
  Outputs: `profiler_enhanced.html`, `profiler_enhanced.json`, `profiler_enhanced.csv`.

## Features & Advanced Topics

- **Live Preview & C Export**: `docs/LIVE_PREVIEW_C_EXPORT.md` – real-time browser updates, embedded firmware headers.
- **Advanced Features**: `ADVANCED_FEATURES.md` – planned or extended capabilities.
- **Animation & Themes**: `ui_animations.py`, `ui_themes.py`, and related JSON exports.
- **Responsive Layout**: `ui_responsive.py` – adaptive sizing strategies.

## Core Source Layout

- **C Source**: `src/` + `sim/main.c` – original implementation.
- **Python Simulator**: `sim_run.py` – primary development target.
- **Designer Engine**: `ui_designer.py`, `ui_designer_pro.py`, `ui_components.py`.

## Remote & Integration

- **RPC Client**: `simctl.py` (quick commands), `esp32_sim_client.py` (library).
- **C/C++ Client Header**: `include/esp32_sim_client.h`.
- **Remote Viewer**: `ui_sim/remote_viewer.html` (WebSocket based).

## Workflow Essentials

- VS Code Tasks (see `SIMULATOR_README.md`): start, smoke, preview, full tests.
- Encoding Reliability: always UTF-8 (`python -X utf8`) to avoid charset issues.
- Artifacts Directory: `reports/` generated by smoke script (PNG/HTML exports + log).
