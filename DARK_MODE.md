# 🌑 Tmavý režim - ESP32 OS Workspace

## ✅ Implementováno!

Workspace je teď **vždy v tmavém režimu** (VS Code styl).

### 🎨 Barevná paleta:

```
Pozadí:           #1e1e1e (tmavě šedá)
Pozadí dark:      #252525 (ještě tmavší)
Pozadí light:     #2d2d2d (trochu světlejší pro tlačítka)

Text:             #cccccc (světle šedý)
Text bright:      #ffffff (bílý pro důležité prvky)

Accent:           #007acc (modrá - VS Code)
Accent hover:     #1e8ad6 (světlejší modrá při najetí myší)

Border:           #3e3e3e (šedá pro rámečky)
Success:          #4ec9b0 (tyrkysová - úspěch)
Warning:          #dcdcaa (žlutá - varování)
```

### 🎨 Stylované komponenty:

- ✅ Hlavní okno (tmavé pozadí)
- ✅ Toolbar (tmavé tlačítka s modrým zvýrazněním)
- ✅ Listbox (tmavý seznam projektů s modrým výběrem)
- ✅ Text oblasti (tmavý text editor pro nápovědu)
- ✅ Status bar (tmavý)
- ✅ Dialog okna (tmavý)
- ✅ LabelFrames (tmavé s modrými titulky)

### 🚀 Spuštění:

```bash
python esp32os_workspace.py
```

Workspace se automaticky spustí v tmavém režimu - **není potřeba nic nastavovat**!

### 💡 Poznámky:

- Barvy jsou inspirované **VS Code Dark Theme**
- Modrá akcent (`#007acc`) je stejná jako v VS Code
- Všechny dialogy a popup okna jsou také tmavé
- Text je dobře čitelný (#cccccc na #1e1e1e pozadí)

---

**Tmavý režim je defaultní a jediný - světlý režim není k dispozici.**
